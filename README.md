## HTTP Crash Course
-HTTP is stateless therefore every request is **independant** 
-Main HTTP Methods: GET, POST, PUT, DELETE 

### HTTP Header Fields
-General: 
 -request URL
 -request method 
 -status code 
  -1xx: Informational
  -2xx: Successful
  -3xx: Redirect
  -4xx: Client Error
  -5xx: Server Error
 -remote address 
 -referrer policy
